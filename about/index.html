This is xrdrsp’s personal website. Updates are irregular.

This website was set up on 2020-01-31. During that time I generated my blog with Hexo and Wordpress. This was shortly before I discarded them and published the present version of this website on 2020-11-01.

Formerly you may leave a comment on the Discussion Board, which is now closed.

I plan to make French and Chinese the primary languages for this website.

Technical Remarks

This blog is currently using the Untitled Theme written by Dovuq and me.

Computer Modern and Noto Sans are used as main typefaces. See also theme.css.

To show mathematics notation with $\LaTeX$, I embedded MathJax 4.0. See also head.html.

The default formatting of the date and time is YYYY-MM-DD HH:MM:SS. To take one example, 2006-11-20 11:40:00 represents forty past eleven, November 20th in year 2006.
